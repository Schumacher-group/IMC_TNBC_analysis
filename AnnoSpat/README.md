# Annospat_labelling
Content description:
- The notebook [post_processing.ipynb](post_processing.ipynb) is the most interesting bit, where the analysis post-labelling is performed.
- Annospat_main contains the AnnoSpat code, it has been lighlty modified from authors' Github [AnnoSpat repo](https://github.com/faryabiLab/AnnoSpat/tree/main/AnnoSpat)
- The notebook [preformatting.ipynb](preformatting.ipynb) concatenates the data from different files into one and stores them in the folder *processed_files*.
  Do not run the preformatting again because it uses files that are not part of this repo.
- The notebook [preformatting.ipynb](preformatting.ipynb) also runs AnnoSpat, feel free to try with different parameters,
 see the Readme of [AnnoSpat repo](https://github.com/faryabiLab/AnnoSpat/blob/main/README.md)
